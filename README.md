# AI Expense Tracker

## Overview

AI Expense Tracker is a machine learning-powered personal finance application that automatically categorizes expenses, tracks spending patterns, generates reports, and provides budget recommendations.

The project uses Natural Language Processing (NLP) and a Naive Bayes classification model to analyze transaction descriptions and classify expenses into categories such as food, transport, shopping, and entertainment.

## Features

- Automatic expense categorization using Machine Learning
- Budget tracking and spending analysis
- Expense distribution visualization
- Category-wise spending reports
- CSV report export
- PDF report generation
- Transaction history management
- Interactive Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-Learn
- Matplotlib
- ReportLab

## Machine Learning

The application uses:

- Count Vectorizer
- Naive Bayes Classification
- Text-based transaction analysis
- Expense category prediction

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
